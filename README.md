# Discount Calculator

A small Python library that applies product discounts (fixed, percentage,
volume) to a shopping cart and returns the total as a money value.

## Requirements

- Python **3.13+**
- [Poetry](https://python-poetry.org/) 2.x

## Getting started

```bash
poetry install
poetry run pytest
```

## Project layout

```
src/discount_calculator/    # library source
tests/                      # pytest test suite
```
